Claude 설치 및 기본 세팅

Claude.ai 가입부터 Projects 설정까지, 브랜드 운영을 위한 기본 환경을 준비합니다.

LevelFoundation
Time20 min
OutputBrand Project setup
Overview Setup Flow Project Settings Practice

Overview

브랜드 운영을 시작하기 전, 먼저 작업 환경을 정리합니다.

Claude는 단순히 질문을 던지는 도구가 아니라, 브랜드의 기준을 저장하고 반복 업무를 함께 처리하는 작업 공간으로 사용할 수 있습니다. 이 수업에서는 가입, 기본 설정, Projects 구조까지 한 번에 정리합니다.

이 수업의 목표는 “AI를 써보기”가 아니라, 매주 반복되는 브랜드 운영 업무를 담을 수 있는 기본 작업실을 만드는 것입니다.

Setup Flow

기본 세팅 순서

01

Claude.ai 가입

브랜드 운영에 사용할 계정으로 가입하고, 업무용 이메일과 연결합니다.

02

작업 언어 정하기

브랜드가 사용하는 기본 언어, 말투, 고객 응대 톤을 한국어 기준으로 정리합니다.

03

Projects 만들기

브랜드별로 Project를 만들고, 콘텐츠 운영 자료를 한 공간에 모읍니다.

Project Settings

Project에는 브랜드 기준을 넣습니다.

Project 이름은 브랜드명 또는 운영 목적이 바로 보이게 작성합니다. 예를 들어 `chaeum / Brand OS`, `ABsent / Content Operation`처럼 사용하면 이후 업무를 찾고 이어가기 쉽습니다.

Project 설명에는 브랜드가 지켜야 할 말투, 핵심 고객, 금지 표현, 콘텐츠 목적을 짧게 넣습니다.

이 Project는 [브랜드명]의 콘텐츠 운영을 위한 작업 공간입니다. 브랜드 톤은 담백하고 전문적이며, 과장된 표현보다 실제 사용 장면과 고객 문제 해결을 우선합니다. 모든 답변은 브랜드의 일관성을 유지하는 방향으로 제안해주세요.

Practice

수업이 끝나기 전 확인할 것

□
업무용 계정으로 Claude.ai에 로그인했다.
□
내 브랜드명으로 Project를 만들었다.
□
브랜드 톤, 고객, 금지 표현을 Project 설명에 넣었다.
□
다음 수업에서 사용할 브랜드 자료 3개를 준비했다.
